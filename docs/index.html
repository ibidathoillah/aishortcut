How They Work

Both auto-installer scripts:

  1. Install prerequisites (curl, git, tar, gzip, unzip)
  2. Install the AI runtime (OpenCode binary or Node.js)
  3. Install & configure the Telegram bot
  4. Set up process management (systemd or background daemon)
  5. Start both services
