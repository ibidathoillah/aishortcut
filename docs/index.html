Deploy an AI-powered Telegram bot in one command

One-Command Setup

bash <(curl -sfL https://raw.githubusercontent.com/ibidathoillah/aishortcut/main/setup-opencode-telegram.sh)

Installs OpenCode AI server + Telegram bot on any Linux machine.

How It Works

The auto-installer script:

  1. Installs prerequisites (curl, git, tar, gzip, unzip)
  2. Downloads the latest OpenCode AI binary
  3. Installs the Bun JavaScript runtime
  4. Clones and builds the Telegram bot
  5. Configures your Telegram bot token & chat ID
  6. Starts both services
